# NASHP Hospital Cost & Transparency Dashboard

## Author
Charles Miller, Texas 2036

## Creation Date
February–March 2026

## Purpose
This dashboard provides an interactive exploration of hospital cost and transparency data published by the National Academy for State Health Policy (NASHP). It enables Texas 2036 policy staff and external stakeholders to examine hospital-level payer mix, operating profit margins, and commercial pricing metrics across all 50 states and D.C. over the 2020–2024 reporting period.

The tool supports rapid policy analysis by allowing users to filter hospitals by state, metro area, urban/rural classification, hospital system, legislative district, bed count, and facility type. A dedicated State Comparison tab enables side-by-side benchmarking of up to four states at a time. A branded variant (TXEAHC-Dashboard.html) was produced for Texas Employers for Affordable Healthcare using their organizational colors and logo.

## Data Sources

| Dataset | Source | Type | Access |
|---|---|---|---|
| NASHP Hospital Cost Tool (Dec 2025) | [nashp.org](https://nashp.org) | Public, aggregated hospital-level data | Public download |
| CMS Hospital General Information | [data.cms.gov](https://data.cms.gov) | Public facility-level data (CCN, ZIP, name, address) | Public download |
| U.S. Census Bureau ZCTA-to-SLDL Relationship File (2024) | [census.gov](https://www.census.gov/programs-surveys/geography.html) | Public geographic crosswalk | Public download |
| U.S. Census Bureau ZCTA-to-SLDU Relationship File (2024) | [census.gov](https://www.census.gov/programs-surveys/geography.html) | Public geographic crosswalk | Public download |
| 89th Texas Legislature membership roster | Texas Legislature Online / Capitol website | Public elected-official data | Public |
| 119th U.S. Congress membership roster | clerk.house.gov / senate.gov | Public elected-official data | Public |

All data sources are public. No identifiable individual-level data, HIPAA/FERPA-protected data, confidential administrative datasets, or internal restricted datasets are used. The dashboard displays only aggregated hospital-level metrics as published by NASHP.

## Technical Architecture
The dashboard is a client-side single-page application with no server dependencies. External libraries are loaded from CDN:

- **Chart.js 4.5.1** — interactive pie charts, scatter plots, and grouped bar charts
- **jsPDF 2.5.1** — browser-side PDF report generation
- **Google Fonts (Montserrat)** — typography

All data is loaded as JavaScript files parsed at page load. No API calls are made at runtime.

## Key Metrics Displayed

| Metric | Field | Description |
|---|---|---|
| Payer mix (6 categories) | p1–p6 | Share of revenue: Charity Care, Uninsured, Medicaid/SCHIP, Medicare, Medicare Advantage, Commercial |
| Operating profit margins (5 payers) | o1–o5 | Medicaid, Medicare, Medicare Advantage, Commercial, and Net Profit margins |
| Commercial Breakeven | cb | Commercial price as a multiple of Medicare needed to break even |
| RAND 5.1 | r5 | RAND-estimated commercial price as a multiple of Medicare |
